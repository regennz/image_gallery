# image_gallery

Flutter plugin to showing all the images from the storage in Android and iOS .

## Usage


```dart
 Future<void> loadImageList() async {
    List allImageTemp;
      allImageTemp = await FlutterGalleryPlugin.getAllImages;


    setState(() {
      this.allImage = allImageTemp;
    });
  }


```
